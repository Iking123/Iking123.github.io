重要的是学会尊重，学会爱
随想 2025/5/31

……最高的目标，是使人学会了解自己善待自己，学会看到他人的苦难，学会爱。

——香港哲学学者周保松

其实我这段想法和周保松没什么关系……

我想的是：作为人，应该学会尊重，学会爱——这很重要，而且似易实难。我这里说的爱是比较广义的爱，但不是指放纵自己沉溺于奶头乐。

也许一切抑郁都源于不够自爱。自爱要像触龙说的“父母之爱”那样——为之计深远；而不是让自己在现代的快感汪洋中随波逐流——那长远来看是自害。

“己欲立而立人，己欲达而达人”——自爱需要自己的“欲立”、“欲达”。真正的自爱，需要自尊，需要自律，需要自强。

想想成功。也许我们可以问：“成功”有什么意义？也许我可以想到，我以前还莫名崖岸自高地鄙夷这个词。但事实是，人也得有起码的成功。

即使我们不在乎名利，我们也得在乎点什么——比如说，假设我们轻千金而重情义。但没有正常人会喜欢废物。正常的废物，也讨厌废物。所以如果我们是个重情义的废物，我们也不会收获我们所憧憬的情义。

个人的成功，当然也需要自爱。我想到了我老师在小测结束后讲的一段话（对课程来说算是跑题的……），可惜发的ppt上没有，截不了图：大概是，他认为，成功需要能力——自主学习、独立解决问题、独立思考的能力，还有与人交流合作的能力，还有坚持、友善与诚实。他还指出，每个人对“成功”的定义不同，在他看来只要一个人有梦想，完成了梦想，对这个人自己来说就是成功。最后还有一句话（小防一手内卷时代焦虑症）：不需要和别人比，只需要和自己比，你今天比昨天更好一点，那你就是最好的。

有志毕生即成功，也无高下也无名。

——老师的“打油诗”

人工神经网络小测
AI 2025/5/31

第1题（单选题）

题目： 关于卷积神经网络（CNN）中卷积核的设计，以下说法正确的是？
选项：
A 对于彩色图像输入（3通道），卷积核的深度可以等于输入通道数（即3）。
B 多个卷积核会生成多个特征图，但所有特征图共享同一组权重参数。
C 卷积核的大小必须与输入特征图的空间尺寸（高度/宽度）一致。
D 使用步长（stride）大于1的卷积操作会显著增加模型参数数量。

答案： A
分析：

  • A正确：CNN里，卷积核深度=输入通道数，卷积核个数=输出通道数。
  • B错误：每个卷积核有独立的权重参数，生成一个独立的特征图；多个卷积核生成多个特征图，但权重不共享。
  • C错误：卷积核大小（如3×3或5×5）通常远小于输入特征图的空间尺寸（如224×224），无需一致。
  • D错误：步长大于1会减少输出特征图的尺寸，但不影响参数数量；参数数量取决于卷积核大小、输入通道数和输出通道数，与步长无关。

第2题（单选题）

题目： 以下关于人工神经网络的说法哪个是不正确的？
选项：
A 反向传播算法基于链式法则计算梯度
B Softmax函数常用于多分类问题的输出层
C 激活函数ReLU比Sigmoid更可能导致出现梯度消失问题
D Adam优化器是自适应学习率优化器

答案： C
分析：

  • C不正确：ReLU激活函数在正区间梯度恒为1，不易导致梯度消失；而Sigmoid在输入值较大时梯度接近0，更容易出现梯度消失问题。因此，ReLU比Sigmoid更不容易导致梯度消失。
